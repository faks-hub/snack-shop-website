Overview

This is my personal portfolio website showcasing my projects, skills, and experience as a Front-End Developer.
The website is built using HTML, CSS, and JavaScript (with plans to add more interactive features later).

Features
-Responsive design for mobile, tablet, and desktop.
-Clean and modern layout for portfolio and contact sections.
-Contact form connected to email.
-Sections for About Me, Projects, Skills, and Contact.

Technologies Used
HTML5 – for structure
CSS3 – for styling and responsive design
JavaScript – for interactivity
